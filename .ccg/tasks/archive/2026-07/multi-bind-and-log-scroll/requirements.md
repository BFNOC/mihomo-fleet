# 支持多监听地址并修复日志滚动

## 需求

- WebUI 默认仍只监听 `127.0.0.1`。
- `-bind` 支持逗号分隔的多个监听地址，例如 `127.0.0.1,192.168.64.1`。
- `-bind all` 或 `-bind 0.0.0.0` 表示监听所有 IPv4 网卡。
- 非回环监听要同步更新 Host 校验，不能因为绑定局域网地址后被现有 loopback-only 校验拦掉。
- 日志自动刷新时，如果用户已经向上滚动查看历史日志，不应每 1.8 秒强制跳回底部；只有当前贴近底部时才自动跟随新日志。

## 约束

- 默认本机安全行为不可变。
- Host 校验仍要防止 DNS 重绑定：只允许 loopback、显式绑定地址，以及 wildcard 绑定时本机实际网卡 IP。
- UI 改动保持安静、紧凑，不引入新的视觉体系。
